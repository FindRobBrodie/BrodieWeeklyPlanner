Lesson 1 - React UI

























App Idea

	-Find a solution to a problem that you have.
	
	-Sketch out idea on a piece of paper.
	
	-Think about what the UI will look like.
	
	-Keep scope narrow at first but will be 
	 a good foundation to for building
	
	
	
	
	















React Components Structure

	-Draw boxes around logically pieces that you can name.
	
	-There should be a natural hierachy forming that will
	 form your component "tree".
	 
	
	
	
	
	
	
	
	
	
	
	
	
	
	
	
	
	
Create New React Project

	-Create React App is a great way to start.
	 https://github.com/facebook/create-react-app
	 
	-I used Visual Studio Community 2019 
		-ASP.NET Core Web Application
		-React.js template
		
	-To follow along, clone/download my repo @
	https://github.com/FindRobBrodie/BrodieWeeklyPlanner	
	
	-Open open VSCode and open folder: /Lesson1_ReactUI/exercise
	
	-npm install
	
	-npm start
	
	
	
	
	
	
	
	
	
	
	
	
Think About State

	-What type of data will app use?
	
	-What component should it "live" in?
	
	-Will state be shared? How will it be passed around?
	
	-React state should be immutable.
